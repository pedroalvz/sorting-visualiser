# Sorting Visualiser

An interactive web app that animates sorting algorithms in real time.
Watch bubble sort and selection sort step through an array — 
comparisons highlight in orange, swaps in red, and sorted positions lock green.

## Live demo
[pedroalvz.github.io/sorting-visualiser](https://pedroalvz.github.io/sorting-visualiser)

## What it does
- Visualises bubble sort and selection sort
- Tracks comparisons and swaps live as sorting happens
- Explains each step in plain English as it runs
- Fully built in vanilla HTML, CSS and JavaScript — no libraries

## What I learned
- How bubble sort and selection sort work under the hood
- async/await in JavaScript to create step-by-step animations
- DOM manipulation — dynamically creating and updating elements
- Big O notation — both algorithms are O(n²)

## How to run it
Just open index.html in any browser. No installs needed.
